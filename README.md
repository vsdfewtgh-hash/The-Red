# TG Short Drama MVP

## 快速启动

```bash
# 后端
cd backend
npm install
node server.js

# 前端
cd frontend
npm install
npm run build
```

## 环境变量 (backend/.env)
```
PORT=3000
BOT_TOKEN=你的TG_BOT_TOKEN
MINI_APP_URL=https://你的域名
```

## 目录结构
- `/backend` - Node.js Express API + Bot
- `/frontend` - React Mini App (Vite)
